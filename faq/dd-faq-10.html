Section 11. A Beginner's Guide to MUDding

[10.1]  How do I get to build an area for the MUD?

Approach an Imm with a proposal for your area. If you haven't much experience, then read the 'Builder's Guide', available on the DD website and as a help entry ("HELP BUILD"). Also read the online help entry for "MOBILE BODY FORM", which contains additional information. Organise how to send your completed area to an Imm, who will then test and link it to the world.

[10.2]  How can I help code the MUD?

Talk to God, and bring your C chops. You'll need to feel at home in a unix shell (probably goes without saying). You'll also need to be prepared to work with code that's been edited by about a dozen people each with different coding styles, can be fairly unstable, and which contains numerous changes which have no significant documentation.

Still want to code?

[10.3]  I want to be an Imm. What do I have to do?

If you ask to be an Imm, you will probably never be an Imm. All current Imms were invited (coerced!) to become Imms, and no player has ever asked to be an Imm and been given the promotion. People who become Imms behave as Imms long before they are called into service.
